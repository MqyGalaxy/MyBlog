# 博客将会暂时保持准停更状态

由于小站站长忙碌于三次元生活学业，小站将在最近暂时进入停止更新状态。在停更状态时，小站的日常维护还是会持续保持，但文章更新将在以下文章发布后停止：

  • 【游戏体验】《少女前线 2：追放》内测评测 （标题待定）

在计划中但暂时停止更新的文章：

  • 【游戏测评】《星空 Starfield》人类永不放弃追逐浪漫且闪耀的群星 （标题待定）
  • Java 从入门到入土 （系列）

# 什么是 “准” 停更

准确来说，小站并不会在此期间一直保持完全停更。也许在某一天小站也会突然更新一篇文章？这可能取决于哪一天稍微比较放松 qwq。
也因此，小站也不希望因为停更而与小伙伴们失去联络。小站也会保持一定的交流状态，在尽可能的时间里回复小伙伴们的留言。

# 小站的未来规划

由于 Shoka 主题年久未能保持维护。小站将于不久之后更换为基于 Shoka 主题二次开发的 ShokaX ，并且将基于该主题进行美化。
在此期间小站可能会进行修复和维护，一部分功能可能因此暂时无法使用。如果因为维护期间出现 bug，请通过联系方式与我联系。

在小站停更期间，如果有友链方面的申请和变动，小站可能无法第一时间进行更新。在此期间我一定会尽力更新友链的最新状态。也请小伙伴们也不要因此嫌弃小站இ௰இ。

如果在停更期间需要与我联系，请通过邮箱直接告知我。评论区可能无法及时回复，通过邮箱我会在第一时间看到您的留言。

感谢各位一直以来对小站的支持！在停更状态结束以后，小站将会恢复正常的更新和交流！

感谢与您相遇～！

2022/11/08

小站最近发现已经有未经允许的商业宣传评论。虽然我个人认为大多数这样的评论估计都是机器人自动发的，但还是声明一下评论区须知：

本站有关发布、传播信息的媒介，包括但不限于评论等功能：

  • 禁止使用于未经允许的商业宣传；
  • 禁止发布引起争议性的评论，包括但不限于议政、歧视、引战等内容；
  • 禁止发布违反相关法律法规、道德风俗的评论；
  • 以上内容也适用于头像、昵称等，能够传播信息的功能。禁止包含恶意链接、二维码、商业邮箱等传播路径；
  • 在相关评论区中，您可以发布有关内容的分享，或者基于此内容的见解和讨论。本站希望评论区能够保持互联网知识分享的价值；
  • 相关内容的讨论和申请，请在其讨论范围以内进行。请勿在其他地方提及。

更多详情请参考小站的评论规则。

小站希望评论区能够保持一个良好的交流环境。小站作为一个非商业性质的个人博客，本身也是一个普通人捣鼓许久的一个作品。希望大家光临小站时，能够保持一个基本的礼节，通过恰当的交流互相了解对方，并且保持大家一个良好的交流环境。

虽然只是我个人的哔哔，估计真发广告的小传单人也不会看。就当是我写了一条说说记录我的博客第一次出现这么多商业广告吧。
