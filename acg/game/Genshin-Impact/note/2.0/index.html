# 2.0（2021/09/02）

主页

# 剧情与探索感悟

2.0 开放已经有几天了，现在我相当是把稻妻给逛了个遍了。
我只能说米哈游属实是把我们给整明白了，整体的设计实在是太巧妙了。

# 剧情走向

在 1.0 期间的原神，旅行者所在的蒙德和璃月大多数给人一种温馨放松的感觉。而且两个国家的神明也为旅行者提供了许多可靠的帮助。
但是一到稻妻，整体剧情变开始倾向另一个方向。旅行者不再受到异国神明的庇护，反而成为了 “永恒” 的敌人。就像是旅行者走出了新手村，终于开始感受世间的险恶。
蒙德和璃月的神一个摸鱼一个在自己的坟头蹦迪也是七个国家里唯二还存活的初代神明。就感觉这样的设计真的挺巧妙的，一开始的神明是你的朋友，但是这次的神明是你的敌人。
在稻妻被胖揍后，偷渡回璃月蒙德像是回到家里了一样 QAQ。北斗大姐头你带我回家把好不好 QAQ

并且稻妻的剧情大多数是比较凄凉的，在雷神一直追求永恒的路上，牺牲多少人们的愿望。旅行者一开始拒绝参与反抗眼狩令的计划，但是在稻妻的旅行中见到了太多悲剧，以及经历了许多不合理的事情。最后与雷神对峙且失败的情况下逃离并且参加了反抗军。

# 探索感悟

米哈游也在尝试更好地利用大世界，将许多信息和剧情内容都塞进了世界任务里。而且一部分世界任务能够更好地帮助你探索大世界。
考虑到多人游戏玩法，也尝试加入了一些小游戏在大世界中。可以在长草时期和朋友消遣。

# 抽卡

# 神里绫华

原本打算把原石全部屯给雷神，结果还是脑子一热把神里抽出来了（゜ー゜）。

# 宵宫

原本打算抽早柚，结果三十发爆了个双黄（宵宫和卢老爷）。雷神有点小慌 QAQ。
