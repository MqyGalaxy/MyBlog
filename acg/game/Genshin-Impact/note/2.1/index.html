# 主线剧情

以下内容包含剧透。如若想要获得完全的游戏体验。请先完成 2.1 魔神任务「千手百眼，天下人间」后再阅读此部分。

# 千手百眼，天下人间

其实很多人都在诟病 2.1 的主线剧情（魔神任务），觉得稻妻的收尾实在是太匆忙了。

mhy 在 2.0 埋下了许多伏笔：
雷神为什么实施的 “眼狩令”？“眼狩令” 对稻妻民众的影响，从一个人失去自己的愿望、自己的亲人，到幕府军与反抗军没日没夜的战争。
旅行者在一开始到达离岛时，就已经感受到 “锁国令” 带来的影响；接触社奉行后，又感受到 “眼狩令” 带来的影响；甚至之后因为被幕府（雷神）通缉，加入了反抗军，了解到反抗军的人们有来自当地不支持 “眼狩令” 的民众、其他地方的受害者 / 神之眼持有者 / 医生等。
而这些伏笔，无一例外地都是为了描写 “雷电将军”（雷神），并且解答她的问题，最后让他转变洗白。
而到了 2.1，mhy 确实是这样做的。整个剧情的重点也在雷电将军的身上。

主线概要
旅行者在加入反抗军后，愚人众以支援反抗军为由倒卖 “邪眼”。旅行者自己的战友哲平，不惜燃烧生命也要使用 “邪眼”，最后衰老离去。
愤怒顿时充满了旅行者的身体（新账和旧账），在反抗军发现了 “邪眼制造工厂” 后，不听珊瑚宫心海的劝阻一个人单枪匹马地杀入工厂，最后被散兵偷袭。
在重要关头，八重神子及时出现，从散兵的手中救下了旅行者用雷神的神之眼与散兵交换。并且告诉旅行者她正在准备一个计划以打破稻妻现在的局面。
八重神子告诉旅行者，现在掌朝的雷神只是雷电将军制造的一模一样的人偶罢了，真正的雷电将军 “影” 则是将自己的意识封入了刀中，进入了 “一心净土”。
由于愚人众的贿赂，除去社奉行的其他两大奉行都在瞒报实情，导致了人偶出现了错误的判断，“眼狩令” 也是这样由来的。目前需要一个人直面真正的雷电将军，说出实情。而旅行者是目前唯一一个能够进入雷神 “一心净土” 的人。
由于在 2.0 被雷神揍得落花流水，八重神子对旅行者进行了一场反雷神试炼虽然通过了试炼也打不过就是了，赠与了旅行者一个御守。
八重神子也尝试说服幕府军大将九条裟罗加入计划，为了找出天领奉行叛变的线索使九条裟罗信服。旅行者一行人找到了早柚，又返回社奉行商量计划，最后成功偷出贿赂的证据。
九条裟罗看到证据之后，在询问天领奉行后，一气之下杀进了天守阁，想要找雷电将军讨一个说法。但是当旅行者追进去时，发现她已经被在场的愚人众执行官女士制服了。
旅行者发现女士与雷电将军（人偶）一行时，愤怒再次充满了身体。将愚人众的罪状统统告诉了雷电将军。但是女士执意声明自己只是一个外交官。旅行者最后便向女士发起了御前决斗。
经历了非常持久且艰难的战斗，旅行者战胜了女士。人偶执行了御前决斗的规则将女士斩杀（无想之一刀）。并且允许旅行者站着离开天守阁。
在离开天守阁的路上，旅行者被刚刚经历的场景感到震惊。正当旅行者走到了天守阁大门前，反抗军的将士和万叶们杀到了天守阁门口。旅行者跨出大门，人偶便拔刀向旅行者砍去真就让我站着离开天守阁啊，一出门就砍。
在此时，万叶与他逝去的友人愿望达成了一致，使友人的神之眼复燃。万叶冲上去挡下了雷神的一刀。
反抗军看到雷神的一刀被人挡下后，一起向雷神冲锋，千手百眼神像中嵌入的神之眼都重新散发着光芒。最后，旅行者再次进入了 “一心净土”。
旅行者将愚人众的阴谋统统告诉了影，影表示旅行者所说的她都知道，并且表示这些事情并不能破坏她正在建造的永恒。旅行者再次向影发起了决斗。
再战不敌，八重神子通过将意识传入御守，最后进入了 “一心净土”，并且提醒旅行者单靠自己是不敌影的。最后旅行者感受到了各地稻妻人民的愿望以及哲平的，最后战胜了影。
影开始深思自己所搭建的道路是否正确，在神子的劝说下解除了眼狩令。

个人感受
2.1 主线剧情的优点就是全程渲染太强了！我第一次玩的时候全程都挺享受的，特别是在天守阁的压抑渲染到万叶挡下雷神的一刀时，整个过程燃到爆炸了。
但是，剥去这个外皮后。仔细品味 2.1 的主线剧情，你会发现完全经不起深度推敲：

首先，反抗军的描写太少了。珊瑚宫心海作为反抗军的领头人 / 军师 / 智力代表，戏份只有一场埋伏战和后面与主角仅有的几次对话。反抗军本身占有的篇幅也十分有限，最后也只是为了引出愚人众在稻妻制造邪眼等一系列阴谋。
作为稻妻主线大伏笔的其中之一，最后的交代太过于草率。
其次就是女士，女士在前期剧情一直都像是一个幕后中 boss 的存在：在蒙德抢走了温迪的神之心，又在璃月和钟离交易拿走了岩神的神之心。再加上圣遗物魔女套对她以前经历的故事描写，后期剧情发力应该会非常不错，甚至合理的话可以洗白。但是最后却惨死在雷神的刀下。
我个人不反感女士的死，生前的大起大落最后却草草死去的角色，渲染到位反而会给玩家一种悲惨的感情。但是在主线整体结尾都很草率的情况下，女士的死反而变得更加廉价了。个人认为女士是因为轻敌，而且被散兵坑了。
更多的就是其他在主线中出现的角色，神里 / 托马 / 宵宫在后期的剧情中就真的只漏了几次脸，北斗等人更是直接没了。再加上先前提到的心海等人，你可以想象他们在事件发生时在忙其他的事情，但是在游戏剧情过程中一次都没有提及过。不是说这样子在意细枝末节太过啰嗦，而是感觉这些角色明明在剧情中，但是与主角的互动太少了。
整个剧情体验时间最多 1.5h 左右，对比 2.0 而言太少了。更何况 2.1 的剧情还是为了填 2.0 挖的坑。

整个剧情，描写的最好的角色：八重神子、影、散兵、旅行者、哲平和万叶。
特别是神子姐姐和影姐姐，太戳我 xp 力 QWQ。
还有哲平哲平我的哲平啊啊啊啊啊啊啊啊啊啊啊 有一种说法就是哲平在生命最后获得了神之眼。

# 心海传说任务

自从 2.1 主线开始拉跨后，大家都将希望寄托在珊瑚宫心海的传说任务上。期待心海的传说任务能够补充设定和完善角色形象。
但结果是心海传说任务并没有改善 2.1 主线剧情拉跨的事实，反而把珊瑚宫心海的人设订得更死了。

# 军师形象崩坏

由于心海在 2.0 时戏份只有一场伏击战，2.1 剧情中反抗军本身也没有多少描写。整体并没有详细表现出心海的用兵之道。
所以传说任务需要快速树立心海作为军师的形象，还要表现出她作为海祈岛领导人的能力。

首先吐槽一下锦囊的设计。我知道所有军师都不是像诸葛亮一样料事如神，也知道使用锦囊是为了表达心海作为军师的远谋。
但是实际剧情里表达的怎么样呢？你可以看到一个反抗军士兵拿出一本堪比字典的锦囊，然后一页一页的翻直到查出怎么解决问题。
整体表现显得非常掉价，感觉整个反抗军像是没有大脑似的，一个包围或者迂回的战术还要查字典才能明白。甚至一些凡事的嘱托也写进了锦囊里。

我认为锦囊应该是在非常危机的时候，拆开用来应急的最后手段；或者是一种非常重要的物品或者信息的载体。
而原神把心海的锦囊描述成一种类似兵法书 / 字典一样的存在，我感觉非常的失败。导致我感觉心海实际上的用兵之道并没有体现出来。

心海在传说任务中有一场实际的指挥：旅行者和心海一同发现了愚人众残余势力的营地，然后心海推理出了三种进攻方案。
但是实际上你还是可以直接冲进去把营地给扬了，而且我更加认为这段更适合表现出作为一个人 / 冒险者 / 士兵的判断，而不是军师这样需要考虑更多因素的位置。

然后就是描写的心海的反差萌，我个人是不反感这样的剧情。但是神里用了一个反差属性，雷神也用了一个反差属性，然后到了心海又用了一个反差属性。
我不知道编剧或者是角色设计是有多喜欢反差。这种设计在第一次神里那里表现的非常新奇，第二次在雷神那里表现的惊讶，但是心海这次主要形象崩塌而且重复使用反差属性反而让人审美疲劳。
在反差中表达了心海她认为自己并不适合作为一位领导者，她更希望自己能作为一名军师。但是军师的人设也站不住脚，导致心海在剧情的定为十分迷惑。

# 剧情

除去宵宫的传说任务，其他三位（神里 / 雷神 / 心海）的传说任务都喜欢用陪逛街来推进剧情，导致部分玩家感到审美疲劳。
剧情上对战后的塑造和描写我认为是挺恰当的。人们虽然有许多遗憾，但是当下最想要的是战争的结束。但是很遗憾的是这并不是整个剧情想要表达的主题。
心海剧情整体没有表达一个确切的的核心讨论点。对比原神之前的传说任务，钟离表达了磨损和对璃月未来的展望，甘雨表达了仙人和璃月人民的关系，可莉表达作为一个孩子的童真……
但是心海剧情整体表现不明所以，原本剧情走向应该是树立心海的个人形象。但是军师形象没有树立好，领导人形象也没有树立，剧情也没有更加深度的主题。最后导致剧情十分无聊。

剧情安排最不适应的还是那部分激进的反抗军，无视上级命令和人民对和平期望私自散播谣言试图再次发起战争。虽然最后误打误撞地抓到了和愚人众有勾结的幕府士兵，但是作为士兵没有接受任何惩罚，反而升职了。
剧情变得更加低龄了，就算原神要考虑自己本身是 12 + 游戏。但是不能压缩自己的剧情质量啊当然不排除编剧本身的问题。至少让那些士兵能够反省自己，而不是一句话带过。
组成秘密部队有一种统一管理加强训练的说法，但是根据整体剧情表现而言，我实在不明白剧情是不是真的是像这样表达。而且我觉得这样只是玩家的脑补罢了，主线并没有明确的提示。

# 周年庆节奏

原神周年庆在周年同人直播的带动下原本非常喜庆。但是在周年当天原神运营方并没有在游戏内给予玩家任何反馈，再加上 2.1 质量的缩水导致大部分玩家的愤怒持续上升。
最后 mhy 补发了四封感谢邮件可能是将原本付费的礼包拆开发布，总共 4 封邮件，包含原石、树脂、翅膀和家具才逐渐平息。

在节奏当中，逐渐从玩家冲策划转移到玩家和玩家的对冲：一部分玩家认为 mhy 免费做的游戏，玩家居然还要奢求 mhy 送礼物；另一部分玩家认为 mhy 逐渐不再关注玩家。
至于我个人的看法：周年庆是感谢玩家陪伴一周年的活动，就算大部分玩家并没有真的陪伴一年的时光，但游戏公司应该给予一些诚意。
mhy 太注重于逐月节、线上音乐会和一部分线下活动了，导致周年庆本身无法再游戏中活动足够的表现。

# 逐月节（重要！）

“逐月节剧情太神了！”

这是我经历了稻妻剧情后再次体验到璃月剧情的感觉。
逐月节吸取了之前海灯节的教训，这次逐月节老角色依次回归客串和互动，还有刻晴香菱和锅巴的全新专属剧情。

# 韶光抚月，人间至味

这次剧情更深度地刻画了璃月各个角色，尤其是刻晴、香菱和锅巴。
无论对待任何事物，刻晴永远都在追随自己的本心。就算在大赛中有自己的朋友参赛，但还是把票投给了自己认为更好的一方，在最后还因为没有给香菱一票还感到非常内疚。
而且在这次剧情中我们还知道刻晴嘴馋金丝虾球，整体刻画得非常可爱又可靠。刻晴作为璃月的玉衡星，更重要的是自己的觉悟和本心，在主线和这次活动剧情就将刻晴的觉悟和本心表达得非常好。

然后就是锅巴，我真的万万没想到锅巴居然有这么庞大的身份。
我直到听到锅巴背后的图标是火属性和岩属性结合的标志，才逐渐猜想锅巴和灶神有关。
我觉得锅巴的 pv 能够直接表达整个剧情对锅巴的描写，因为这是我的言语无法传达的感动：

璃月曾经是由帝君和众多仙人守护着。如今他们逐渐离去，甘雨、刻晴、凝光、北斗、香菱等许多璃月的新秀接管仙人们的责任。如今璃月不再需要仙人保护，不再需要仙人的馈赠，人们会踏着这片土地一直走下去。

“风雨犹祝，山海同欢，是承天地之佑。星移斗转，沧海桑田，烟火人间依旧。”
“功名在我，百岁千秋，无忘秉烛夜游。今古诸事，激荡中流，宏图待看新秀。”

璃月送别了保护自己的魔神，但是在未来，会有更多的新星去守护这片土地的美好。

我推荐自己观赏逐月节的剧情，于是我就不写剧情概要啦～
