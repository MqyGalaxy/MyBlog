# 原神生日会（2021/09/28）

主页・我的周年剧场

# 直播回放

推荐前往 生日会 页面或者 直播回放 观看！

# 感触

这次的生日会直播质量简直就是爆炸，完完全全可以感受到玩家对这款游戏的喜爱。
我一直仍为一款游戏或者 ip，如果本身质量过硬，再加上给玩家合理的二创氛围，那么这款游戏所引起的讨论度肯定很高。
前有东方、Fate、碧蓝航线，后有明日方舟、原神。他们都拥有着非常良好的二创氛围，所以我非常喜欢这些游戏。二次创作带来的内容和玩家们的讨论，让整个直播就像庆典一样，属于玩家的狂欢。

在这次生日会里，我特别喜欢以下全部节目：

# 生日会目录

# 0:00:00「可莉的邀约」——M 有病

直播待机画面。

# 0:04:57「繁星闪耀之序章」—— 姬海棠羽笠果

同人粉丝创作的 OP。

# 0:07:38「提瓦特漂流赛！」—— 拉环儿_

生草整活小短片。全程笑着看的。

# 0:10:23「Butterfly」——BenAB

芭芭拉冲吖！

# 0:11:54「天地四方」—— 暗猫の祝福

钟离好帅！非常有中国风特色的创作。

# 0:14:43「记忆的回声」——oskarpianist

这个是我认为的全片最感动的片段之一，全球不同地区的原神玩家在一起演奏了原神的音乐。
全球的玩家能享受一款游戏带来的乐趣。

# 0:20:51「在璃月」—— 小虾鱼 Official

逐月节刻晴和香菱的主场！

# 0:24:12「关于胡桃想把大伟丘愉快送走的那档事」—— 神奇的老皮 VFX

老皮的生草特效短片。

# 0:30:12「风尘苦旅」——KKOMDA

K 佬也是一个 Youtube 入驻 bilibili 的二创作者，画风是真的可爱。

# 0:32:21「舞刃谐焰曲」——- 麦粒太太 - Ver-

特别燃的一段动画短片，画面帧数分镜打斗都让人感到燃起来了我打这个 boss 的时候怎么都没这么燃 qwq。
原宝赶紧出番啊！！！！！

# 0:36:27「逐君千里，月印同心」—— 哈米伦的弄笛者

算是一个生草小剧场了吧，屑旅行者筹办逐月节海灯节的海灯都是爷一个人编的。
哦对了，还有考哥.jpg。

# 0:44:02「原神旅行日记」—— 渊渊的奇妙冒险

这也是我非常喜欢的片段之一，非常好听带感的一个音乐剧！
屑旅行者一年的旅途。

# 0:55:38「神里绫华的夏日大作战」——JordanTseng

一个看到都令人肝痛的定格动画。

# 0:58:48「钟离大战达达利亚」—— 影视飓风

铠甲勇士帝皇侠钟离 大战 假面骑士亚极陀公子
也是非常厉害和许多 neta 的特效短片。

# 1:06:20「驻足在这星空下」—— 梦鼠计划

非常好听的 MV 里面的皮肤可以考虑一下吗 mhy（滑稽）。

# 1:10:05「原来是这样解决的吗？真是太…！」—— 果蝇轰

生草闯关短片， #原神小剧场# 。

# 1:17:00「如露亦如电，如梦幻泡影。」—— 暗猫の祝福

我一直都很喜欢暗猫大佬制作的手书。2.1 能够深挖的内容太多了，而大佬把细节全部挖了出来，接上了之前制作的万叶手书，主要描写了万叶和影（雷电将军）的过往和经历，直到最后万叶接下了无想的一刀。
影在战争中斩杀了海祈岛的魔神，也目睹了自己挚友和血亲的离去。为了消灭痛苦，她走上追寻永恒的道路。
万叶的友人不认同雷神在追求永恒时过激的作为，与雷神决斗时战败，失去了自己的生命。万叶带着他熄灭的神之眼匆忙逃亡。
当万叶见到又一位友人即将被雷神斩杀时，愿望重新点亮了神之眼。

「总会有地上的生灵，敢于直面雷电的威光」

万叶完成了友人的愿望，影也打开了自己的内心。

# 1:22:25「外乡人 2」—— 不死な千咲

没想到有生之年等待更新的系列会出现在生日会。
在「外乡人 2」中，你可以看到生草，neta，梗，欢乐，回忆和刀。
《当 90 级旅行者看到了当时 1 级的自己的黑历史》
总之，看就完了！！！！！！！！

「外乡人始终还是是外乡人，你还记得家的味道吗？哥哥。」
「但是我们的旅途中总会有陪伴着我们的朋友和回忆，我们终将重逢。」
「无论你当前的境遇如何，提瓦特大陆的星空永远存在着你的位置。」

# 1:36:53「旅途」—— 暗猫の祝福

非常精致的剪辑。我们终将重逢，但不是这里，不是现在……
简直就像是原神官方宣传片。

# 1:41:14「门的另一端」——YuH_ChiliChill

辛苦了！旅行者，大家在门的另一端等着你。

# 1:45:35「提瓦特战线」—— 帷幕 VMovie

血界战线描改。
原神一周年的回忆，下一个一年见！

# 1:48:03「彩蛋」—— 主创团队

幸苦了大佬们！
