# 前瞻直播

  • 新角色 八重神子 为我们进行了稻妻部分区域的前瞻讲解。八重樱？！
  • 公布了 优菈 和 烟绯 将在 1.5 登场。
  • 两个祈愿池 陵薮市朝 (钟离复刻) 和 浪涌之瞬 。
  • 1.5 版本中 2 个角色祈愿池分别为：钟离和烟绯 (陵薮市朝)、优菈 (浪涌之瞬)。
  • 新的武器 松藾响起之时 和圣遗物 千岩牢固 与 苍白之火 。
  • 新的怪物 骆驼龙王 、 无相之冰 、 深渊咏者·紫电 。
  • 新系统 尘歌壶 (家园系统)。
  • 邀约任务第二辑 迪奥娜第一幕 与 诺艾尔第二幕 。
  • 新的传说任务： 钟离第二章 和 优菈第一章 。
  • 新活动：
    • 导能原盘・绪论
    • 精通移涌
    • 风行迷踪（躲猫猫，好耶！）
  • 优化：
    • 每周前三次挑战大 BOSS 收取奖励消耗树脂减半。
    • 多人联机状态下好感度获取加倍。
    • 限时活动结束前提醒邮件。
    • 系统设置可自由管理多语言语音包。
