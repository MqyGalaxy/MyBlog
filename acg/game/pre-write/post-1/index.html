MOEGN 评测

《少女前线2：追放》是基于前作《少女前线》剧情十年之后展开的后续故事。在游戏中独特的美术风格、简洁的HUD、以及精细的人物建模给玩家带来了愉悦的第一印象。但尽管本作有着高质量的美术设计与高人气的前作基底，《少女前线2：追放》却给玩家们带来了一个既不符合商业逻辑，也不符合系列粉丝期待，充满了一股扭曲自傲的恶心产物。无论是本作游戏剧情与前作的割裂，还是指挥官与战术人形之间的人机关系断裂，甚至在剧情中对玩家恶意遗留的情感基调，无一不在透露制作方对前作玩家群体的漠视。游戏剧情的文本剧情和活动策划缺乏现实生活中的基本逻辑，更是缺乏对用户的基本尊重，就像只是为了满足个人喜好而诞生出来的一样。即便如此，散爆网络却依旧对游戏存在的主要问题避而不谈，用转移矛盾的方式忽悠玩家群体。最终使得《少女前线2：追放》注定只能成为一款脱离玩家的“自嗨”游戏，也让《少女前线》这个IP雪上加霜。
