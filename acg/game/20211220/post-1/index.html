# 前言

我最开始关注到幻塔，是当时的一个游戏开发视频。让我知道除了原神，还有人也在开发二次元 “开放世界冒险” 游戏。
当时的幻塔虽然漏洞很多，但是玩家们也并没有太苛刻要求。毕竟这是一个年轻工作室刚开发的作品，我们详细他们能够越做越好。

我也尝试过抢幻塔的内测资格，但很可惜都没有选中。

然后直到今年的 12 月 16 日，幻塔开启了全平台公测。我终于可以自己尝试体验幻塔带来的世界。

# 开幕雷击

首先话先说在前面，《幻塔》给人带来的体验并不是沉浸式开放世界冒险，而是一款完美祖传 MMORPG。

我找到了幻塔的官方网站后，想要下载幻塔的 PC 端。但是我找了半天都没有找到 “幻塔 PC 端下载” 的字样。
结果最后告诉我，那个叫 “幻塔手游助手” 的下载按钮就是幻塔 PC 端。

首先就不吐槽这个使人迷惑的名称了，听说幻塔 PC 端由于没有批准版号所以才会变成这样。但 是 为什么 PC 端必须使用幻塔手机端扫描登陆？
我想在 PC 端玩幻塔，我不但要下一个 “完美手游助手”，还要下一个 12GB 的幻塔 “手游”；最后还要在我的手机上下载 5GB 以上的幻塔手游 “登陆器”。
然后每次登陆游戏，我还要把手机的幻塔打开，然后用它扫描电脑幻塔的二维码。我上一次见到这么迷惑的登陆操作还是在微信。

幻塔好心的是在开服前开放了 “预捏脸” 的功能，可以让玩家提前捏角色。并且自己捏的角色可以分享给其他玩家。

然后就是进入游戏了，首先就让我看到了网游祖传分服：什么 “星岛 HT-01”“XXX-02” 啊。这好歹还是一款 2021 年末的网游了吧，不至于还把服务器分的这么多来滚服吧。
说真的我一直认为幻塔会选择 “地图分服”：玩家匹配在不同的大世界。可以参考一下 GTAV 的战局和星际战甲的大世界。但是这种滚服设计能在 2021 年见到基本让人感觉是卷钱游戏，第一印象直线下降。
进入服务器，又会看到一个祖传的爆满排队。不知道是你的服务器真的小还是想学习隔壁网易。

# 进入游戏

网上很多人已经在终测时评测过幻塔了，很显然幻塔依旧充满着许多漏洞。但是终测最后能修改的东西肯定不多。
公测开始后，更加印证了我的想法。但是令人欣慰的是他们确实是修复了许多 bug 和人物姿势：比如人物的跑步姿势、攀爬姿势、游泳的阻力、载具 bug 等影响较大的因素。
而且在 PC 端中，画质和打击感虽然不算顶级，但已经到及格的位置了。但是手机端就并不怎么样了，手机端的画质和打击感不仅大打折扣，而付出了这么大的代价换来的优化也微乎其微。

在进入游戏的几个小时，幻塔确实给我带来了许多惊喜。比如幻塔想要描述的废土世界观和他们带来的崭新的画风。
他们知道自己能力有限，于是他们也学会了缝合多种玩法。我并不讨厌缝合，如果一款游戏缝合到了极致反而会吸引玩家（例如《创世理想乡》）。
缝合的内容包括了解密、战斗、大世界交互等。这的确能给游戏带来更多的可玩性。

# 各种问题

幻塔一开始带来的体验非常不错，但是玩久了带来的却是疲劳。

# 迷惑剧情

二次元游戏大多数还是以养成为主，其养成的动力来自与世界观的架构以及角色的塑造。
我可以完完全全的告诉你，幻塔里的剧情和角色塑造：简直就是一盘由双料高级特工缝合出来的屎。

首先主角死里逃生，最后被反派偷袭昏迷失忆。星岛避难所的夏佐和他的妹妹发现了主角并把他 / 她带回避难所。并且治好了主角身上的伤以及修复了主角的抑制器。
在这个世界，人们需要使用抑制器来保护自己不受某种 “辐射” 的侵害。如果抑制器损坏或者丢失那么人们就会被感染，最后变成行尸走肉。
妹妹带着主角介绍避难所，并且带着主角做了很多事情：比如修复避难所的折跃门和破解遗迹的大门。
但是之后不知道为什么避难所遭受匪徒袭击，妹妹在途中为了保护战斗中被偷袭的哥哥夏佐，抑制器遭受了损坏。虽然经过了紧急治疗但是还是被感染了。
途中妹妹拖着病重的身体，独自离开避难所。想要安静的欣赏夜空，最后被主角发现。妹妹说自己非常喜欢这个地方，小时候遇到不愉快的事情就会跑到这里来和哥哥仰望星空。
但是后来妹妹的身体还是没能撑住，身体逐渐虚弱。哥哥希望主角寻找帮手帮手就是一开始袭击主角的反派，并且告诉主角这个帮手是他们的秘密，希望不要告诉其他人。
主角找到了帮手所住的地方，但是建筑里早已人去楼空。剩下一台还在运行的电子设备。幸运的是主角联系上了帮手，但是对方仅是草草地回复了主角。
回到避难所后，妹妹已经开始尸化。避难所的人们催促着哥哥处决妹妹，但是最后哥哥没有处决，带着妹妹离开了避难所。最后他们被帮手带走，帮手声称他们能够拯救妹妹。于此同时，所有大陆的能量塔发送了异常现象。

在幻塔大陆，中心有一座巨大的塔 “海嘉徳”，在那里居住的许多科研人员和高层。他们会在每年向不同地区的能量塔输入能量，一方面保证其他地区的能源补给，另一方面能够保证 “海嘉徳” 的能量不会过载，从而避免大范围的爆炸和辐射。
但如今所有的能量塔开始反向输出能量传回海嘉徳，能源过载的危机影响着整篇大陆的未来。主角来到星岛的能量塔，并且通过折跃门成功登上了海嘉徳。了解到自己之前曾是海嘉徳的执行员。今后主角就踏上了关闭能量塔和寻找兄妹两的道路。

听完我的描述后，是不是觉得幻塔剧情还不错？那是因为这是我经过思考后重新排版总结出来的，和幻塔原本描述的剧情相似的剧情。幻塔剧情的演出表现迷惑的地方真的太多了，完全无法将它完整地写出来。
首先我完完全全地想吐槽剧情里的兄妹两人。就不说基本剧情推进全程就是妹妹拉着你跑，然后妹妹惹事主角收拾烂摊子。到后面避难所大多数人要求处决妹妹，哥哥直接拒绝的操作：一开始看感觉挺感动的，但是后面解释处决的规矩是哥哥夏佐自己定的。
自己定的规矩自己不遵守，这难道不是顶级双标吗？然后反手就加入了反派，我还要全大陆跑找你们俩。全程没有任何感情铺垫，我为啥要去找你们。我连现在妹妹的名字是啥我都忘了，妹妹还是这个游戏的看板娘啊，一点让玩家记忆深刻的剧情都没有就领盒饭了。

更别说后面可有可无的剧情。在海嘉徳拆炸弹时所使用的时间回溯，提了一嘴就没了。我真不明白这一段是为了所谓的铺垫还是纯粹的拖时间。

# 拟态角色

因为幻塔的卖点就是抽武器，在武器上贴给人皮吸引玩家去抽取。
但是我可以告诉你，我完全没有任何欲望去抽。尤其是我做完第一个拟态任务后。

拟态任务相当于某游的传说任务。是补充角色背景和人物塑造的机会，是展现角色的特点和玩家的互动。
但是幻塔的拟态任务，我完全感受不到。本身建模质量就是各种要素往上堆就算了，人物完全没有特点。互动更没有，因为全程都是让你去回收他们的碎片，然后偷窥别人的记忆。
第一个拟态任务我就只记得她弟弟被围攻失踪，她找了一辈子。没了，整个剧情就为了讲这件事，玩家就是全程看戏。
拟态任务和其本身主线剧情一样，毫无铺垫就给你来一刀。我真不知道这样子怎么能打动玩家？怎么能吸引玩家抽取角色。
拟态角色还不如我捏的好看

# 大世界、工会、爬塔、排名

任务做完了，至少让我探索一下大世界吧。
幻塔也有四个每日任务，但是基本上都是刷怪。但是每日任务的怪都是大世界一起刷的，和其他玩家共享。也就是说其他玩家也可以抢了你的怪。导致你还必须站在原地等下一次更新。
宝箱、解密、素材等也是和其他玩家共享的，如果来晚了也只能等待刷新。
然后就是世界频道齐刷刷的喊工会招人、打花。说实话那个时候我连花是什么都不知道。
然后就是排名，什么等级排名、成就排名等都给你齐刷刷的安排上了。
幻塔太注重于社交了，导致根本无法带来沉浸式的开放世界体验。

# 抽卡、福利

幻塔保底 80 抽，但是基本上都是快抽到 80 抽才出的。
并且幻塔分为黑核、金核以及未知的红核（红核是玩家卡 bug 发现的限定池，目前没有获得途径）。
也就是说，黑核是友情池、金核是普通池，而红核才是限定池。
虽然大世界满地的金核，但金核也只是普通池的门票了。

至于幻塔福利，目前还只有开服的免费金核十连（还是炸服给的）。

# 游玩体验

幻塔本身质量能超过其他友商粗制滥造的批量生产游戏，但是要和原神对比简直差了一万条街。
为什么要把一款 MMORPG 和开放世界游戏做对比，因为幻塔本身宣发就是 “下一个二次元开放世界冒险手游”，其出现的各种宣发操作，甚至开小号都是在贴脸原神。
甚至在开服时他们的广告是 “下一个 TGA 年度手游”，配吗？一个半成品游戏配吗？
不得不说本体质量确实够硬，但是没有二次元味。仅仅就是一款二次元套皮手游。角色完全没有吸引力，剧情更是低龄到极致。
我一开始也希望幻塔能够打出自己的一片天下，但是他在宣发上搞出了一堆迷惑行为，最后吸引大量玩家来看戏，结果带来的却是这样的体验。

······

如果是为了寻找一款二次元开放世界冒险，幻塔无法带给你最佳的游戏体验。但是如果是为了寻找刀剑传奇的替代品，也许幻塔能够给你一种崭新的体验吧。
