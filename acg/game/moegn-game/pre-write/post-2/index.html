MOEGN 评测

《绝区零》是由米哈游全新研发和推出的动作游戏。人们生活在一个遭受神秘且危险的超自然灾害“空洞”侵袭的近未来世界中，由于空洞灾害导致旧都“艾利都”被毁，许多人们的生活轨迹从此发生变化。面对空洞侵蚀以及其中隐藏的种种怪物和危机，人们建立了“新艾利都”，作为人类最后的绿洲开始研究与空洞共生的技术，而在这座新兴且活力的城市中各种错综复杂的不同派系开始环绕着空洞所蕴含的未知力量展开争斗。作为一名传奇“绳匠”的玩家将不得不在这座都市中面临种种抉择，在危机四伏的空洞中引导着“代理人”们冲破迷途，并在“新艾利都”中感受多样且丰富的都市生活。游戏中大量的渲染动画让整个游戏过程非常丰满，配合绝区零本身超高的风格化美术以及创新的平面设计美化超越了目前美术同质化极其严重的国产游戏。但是枯燥的游戏模式，平庸的动作系统，以及依旧换汤不换药的养成系统，搭配上可以称之为无底洞的米氏抽卡和未来可期的数值膨胀，使得《绝区零》即使美术风格再怎么丰富，也注定无法成为一款“好玩”的游戏。
