# 更新主题：洞穴与峭壁 Caves & Cliffs

1.17 预定推出时间为 2021 年，与以往 1.16/1.15 版本不同，更新时间往往大于它们
所以 1.17 目前推出的内容可能只有一半，是一个跨度 1 年的大更新

1.17 官方预定更新时间为 2021 年夏天

首先是对应了主题，对洞穴 / 高山的地形生成进行了大更新
洞穴群系更新了：繁茂洞穴 石笋洞穴，而高山则变得更加真实雄伟
也就是说，本次更新对于主世界 y 轴的探索进行了深造

# 高山内容更新

# 新的生成机制

新地形
高山的生成将会更加真实雄伟

雪块
玩家会陷入雪块中

# 新的生物

山羊
跳跃能力强，能跳很高的距离
会攻击生物，会造成一段击退效果

# 洞穴群系更新

# 新的生成机制

洞穴
洞穴将生成的更大，更加的空旷

地下湖
在洞穴中会形成一些湖泊，你可以在上面划船，可能也会填充整个矿洞

# 繁茂洞穴

植物繁茂，花草丛生，会生成许多新植被，在地上会有一个小洞口直连洞穴

发光浆果、孢子花朵、杜鹃花
未知，但发光浆果作为整个洞穴的光源

浮萍植物
玩家可以站在该植物上（该植物顶部有碰撞体积），但站一段时间顶部会慢慢倾斜，最后玩家会掉下来（失去碰撞体积）

# 石笋洞穴

会分布 石笋 与 钟乳石（对应现实中喀斯特地貌）

石笋
会滴水，并且可以使用容器收集

钟乳石
会对生物造成刺伤，破坏根部会坠落，生长在矿洞顶部或底部

# 紫水晶洞

非常稀有，可以开采水晶碎片

# 其他（植被 / 生物 / 矿石 / 道具）

含羞植被
会对声音（震动）产生反应，可以用于制作含羞侦测器，引发红石信号，也可以使用多个以达到无线传输，貌似可以使用羊毛隔音

循声守卫
会生成在很深的地下，听声辩位（震动 / 无法用视力检测玩家），攻击力很高（下界合金套也很难应对），已知为敌对生物

蝾螈
友好生物，洞穴水中的新生物，可以用水桶舀起
被桶装后放出就会跟随玩家，帮助玩家在水下攻击敌对生物

铜矿
新矿物，可炼铜炼！都可以炼！可制作铜块、避雷针等
铜在放置后会生成铜锈（青色？）随着时间变化越来越多直到整个方块

# 考古系统

# 玩法

由于介绍不够全面，这里只列出大部分玩法

遗迹
玩家可以在遗迹中考古挖掘矿石与古物

毛刷
玩家不能直接破坏方块，需要使用毛刷从泥土、沙砾中筛出古物与矿石

陶罐
玩家可以将古物粘在陶罐上，之后便可烧制成品

# 其他

# 生物

发光鱿鱼
海底生物，移动光源，具有催眠效果（表现未知）
为投票胜出生物

# 物品

望远镜
与高清修复 mod 的 C 键用途相似，视角会有圆形黑边框，使用水晶碎片合成

避雷针
吸引雷电，能防止雷电劈中易燃物质（木制、羊毛等）
其他用途待开发

口袋
可以收纳背包内的物品，可以堆叠总共 64 个不同物品（类似背包 mod）
潜影盒直呼内行

# 最后

这次的洞穴更新更加丰富了 MC 的地底世界，而这正是玩家想要看到的，可以说，这次的更新是民意的胜利了
