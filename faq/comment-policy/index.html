# 用户守则

# 用户

“用户” 指的是浏览或使用本站评论系统的个人或组织。

# 个人隐私

请尊重每一个在本站留言的用户。本站禁止发布涉嫌有关跟踪、威胁、骚扰、胁迫、侵犯用户个人隐私、披露用户个人信息等内容。我们将严肃处理违反上述和其他规则的行为。

# 恶意冒充他人

请勿通过昵称、头像、网站链接等恶意模仿其他用户（或者其他未使用本站评论系统的个人或组织），并伪装模仿的对象与他人进行互动。这不仅是为了防止对其他用户造成一定的误导，也是为了尊重其他个人和组织的身份和隐私。

# 欺诈、诈骗等行为

请勿伪造信息（例如伪造个人 / 组织信息、网站链接、活动内容等）以为自己谋取利益。我们会删除这些故意误导用户的内容，并向上传者发出警示（详见惩罚措施）。

# 恶意言论

MOEQY 希望用户在评论区能够维护和保持积极、友善的交流和互动，鼓励发布个人的意见和观点。但我们不允许用户发表包含涉及种族、宗教信仰、性别、年龄、国籍、残障、性取向、政治倾向等，针对特定对象或群体的语言人身攻击，禁止籍由以上内容煽动对立、仇恨情绪。

请勿对任何个人或组织进行不正当的嘲讽、发布仇恨言论、挑拨他人矛盾。我们会删除这些恶意攻击的内容，并向上传者发出警示（详见惩罚措施）。

# 垃圾内容

为了保持评论区的和谐、规范，以及保持评论区良好的交流环境。我们会定期清理无意义的垃圾评论。

我们对 “垃圾评论” 的界定：

  • 与文章内容无关的；
  • 与讨论范围无关的；
  • 无意义灌水的（例如意义不明的乱码、表情、颜文字等）；
  • 不遵守相关规则的；
  • 涉嫌 “违禁事项” 的（详见违禁事项）；
  • 违反 “用户守则” 的。

# 违禁事项

当您在本站中发表言论时，包括但不限于：

  • 本站禁止发布违反法律法规、道德风尚的言论；
  • 请勿在评论中写有带入任何政治倾向的言论；MOEQY 尊重所有人的政治立场、想法，但是 MOEQY 并不是发布您的政治观点的地方；
  • 禁止发布带有对某个个人、组织的不正当侮辱性言论；
  • 禁止发布带有对某个种族、民族、性别等偏见歧视性质的言论；
  • 禁止发布带有未经允许的商业性质宣传（包括但不限于链接 / 头像 / 昵称 / 二维码 / 图像 / 视频 / 邮箱 / 电话号码等）；
  • 禁止发布带有不良内容宣传（包括但不限于链接 / 头像 / 昵称 / 二维码 / 图像 / 视频 / 邮箱 / 电话号码等）；
  • 禁止发布欺诈、诈骗等内容（包括但不限于链接 / 头像 / 昵称 / 二维码 / 图像 / 视频 / 邮箱 / 电话号码等）；
  • 禁止发布带有容易引起生理性不适的言论（包括但不限于链接 / 头像 / 昵称 / 二维码 / 图像 / 视频等）；
  • 禁止冒充他人、利用他人的名义发布任何信息。

本站不会承担您违反守则言论所造成的相应责任。我们会经过举报、筛查以及第三方请求，删除不符合守则的言论。
