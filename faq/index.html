评论规则

小站欢迎大家积极讨论交流！但请注意以下事项：

  • 本站禁止发布违反法律法规、道德风尚的言论；
  • 请勿在评论中写有带政治倾向的言论：本站尊重所有人的政治立场、想法，但是本站并不是发布您的政治观点的地方；
  • 请仔细思考再发出评论，本站禁止发布垃圾评论，包括但不限于：
    • 带有对某个组织 / 个人的不正当侮辱性言论；
    • 带有对某个民族 / 种族 / 性别等偏见歧视性质的言论；
    • 带有未经允许的商业性质宣传（包括但不限于链接 / 头像 / 昵称 / 二维码 / 图像 / 视频 / 邮箱 / 电话号码等）；
    • 带有不良内容宣传（包括但不限于链接 / 头像 / 昵称 / 二维码 / 图像 / 视频 / 邮箱 / 电话号码等）；
    • 带有容易引起生理性不适的言论（包括但不限于链接 / 头像 / 昵称 / 二维码 / 图像 / 视频等）；
  • 关于添加友链的评论详见 添加友链 ；
  • 关于 “留言” 页面详见 留言 ；

如若违反以上规则将会删除并给予警告，多次违反将列入黑名单。
本站会经过举报、筛查以及第三方请求，删除不符合评论规则的评论 / 留言。最终解释权归本站所有。

添加友链

欢迎各位小伙伴找我玩！

以下是交换友链的方式以及注意事项

  • 在原则上，本站友链欢迎各类网站（包括但不限于博客 / 同人创作网站主页等），但不接受拥有任何违反相关法律法规、道德风尚内容的网站。
  • 欢迎所有博客类型，无论您是正在学习的新手，还是各种神仙大佬，本站都会收入友链。
  • 本站友链采取互换制，除去特殊情况，如果您将我的友链撤下，相应的我也会把您的友链撤下。特殊情况请在评论区回复。
  • 如果您的网站无法访问，本站并不会第一时间撤下，而是将您的友链特殊标记，如果您的网站可以再次访问，请在友链评论区回复。
  • 为了保持网站规范，以及方便友链的添加，请您不要在除友链外其他页面评论区发布友链请求。
  • 友链添加成功后，如果您在评论时注明了邮箱，您可以在评论区获得” 老婆！”（waifu）的标签。
    * 如果您未能获得友链标签，请在留言版留言或通过联系方式联系我。
  • 本条例可能不定期进行修改，但不会影响已经添加的友链。
