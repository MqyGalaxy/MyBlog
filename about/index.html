# 建站经历

一开始看到了其他大佬的博客，一瞬间就对搭建个人博客感兴趣了对，一瞬间就感兴趣了。于是自己就开始马马虎虎地就开始摸索了。
作为当时代码基础为 0 的废物萌新，在 b 站上找到了个用 <table> 标签写网站的视频，然后装模做样地写出了博客第一个版本。现在返回去看我留下的网页，真的开始怀疑自己的审美了。
当时的我也发现了这个毛病，于是又开始摸爬滚打地写出来了第二个版本。第二个版本在美观的角度上挺不错，但是由于是边摸边写的，结果连自己都看不懂自己在写什么了。
之后就了解到了博客框架，当时全手动更新的我发现使用博客框架不但效率高，还可以找一些大佬设计的相应主题。又在一系列摸爬滚打中最后看中了 hexo，并且认识了 shoka 这个漂亮的主题。
目前这个主题是我找到的最实用的主题，写文章时可以使用的特殊功能有很多，而且还能保持整体的简洁。如果以后自己写主题的时候也会考虑参考这个主题。
以上就是我建立博客的经历了，大家就当看个乐子就行了。 ( •̀ ω •́ )✧
