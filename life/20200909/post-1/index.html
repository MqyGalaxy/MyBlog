这里是 MqyGalaxy~

虽然这个博客一是静态，二没有后台，但我还是会在这里写一些东西（没有交互功能应该也没人看吧 QAQ）

在这篇博文中，我会说明接下来做为 UP 的走向以及之后的安排

# 为什么几个月没有更新视频了？

在这里我要向大家解释一下：

首先：在最近实在有自己重要的事情要忙，更新视频没有余力
其次：最近在对前端制作下功夫，由于本人对此领域毫无研究，还在一步步的摸索
由于现实生活和自己的兴趣而耽误了视频制作，虽然我的粉丝少到可怜，但还是要在这里向关注我的人道歉！

# 今后的视频内容走向

以下是我今后的视频内容走向：

  • 暂时减少制作游戏区视频
  • 动画 MAD 制作
  • 番剧类视频
  • 生草视频
  • 东方 Project 视频

因为个人以及他人的安排不同，很难像以前一样聚在一起打游戏录视频了，再加上现在本人所用的笔记本也带不动一些高级大作，游戏区的视频肯定会比以前少很多

因为对于媒体制作的爱好，从以前就已经开始剪 mad 了（虽然很菜，当然不排除本身硬件问题），会逐步剪辑更上一层楼的视频

都是老二刺螈了，不出些二刺螈的视频怎么行？？？？

个人也喜欢生草整活，最近比较生草的片段都会拿来整活的

虽然东方也是游戏，但其同人创作涉及范围广而且我是车车人，平常也会制作一些与东方有关的视频

# 今后自身的安排

  • 学业第一，如果爱好与学业冲突，非过激情况会优先选择学业
  • 由于最近个人的现实生活，不会保证视频的更新内容与速度
  • 到目前为止，除非本人今后的需求，目前制作的视频等内容仅为自身爱好，不强求观众充电 / 打钱
  • 学习前端制作，创建属于自己真正的个人博客
  • 关于非过激情况，目前我也说不清楚，但我不会轻易放弃自己的爱好！

我的视频不定期更新，也是因为自己的现实生活与视频创意 / 视频质量等原因，希望观众见谅！

我目前并不建议观众朋友给我充电或者是送礼，做视频就是为了爱好，不会强求收钱收礼

最后这个目标主要也是因为自己的爱好，有机会也会与大家谈谈

# 什么时候更新新视频

希腊奶～

# 以前的视频还会填坑吗？

看情况，如果是远古巨坑（无法填）或者是黑历史，我可能会看情况砍掉

# 为什么到现在也要坚持下去？

就像这个网站，就算没人看也会写下去，为的就是和你一样正在阅读的观众

我做视频从来也没求过三连，有就投币没就点赞，只要你能点赞让更多的人看到，我就已经很开心了

东方有一位制作同人游戏的大佬，他说做游戏的初衷就是为了满足自己的创作欲望，就是想做。我也一样，只要有人看到、有人点赞、有人评论、有人收藏，对于制作者心里肯定是很开心的

# 最后

现在我会坚持制作视频和编写博文，博客的内容也是制作视频的技巧或者 ACG 内容（技术还得看将来）

欢迎大家来我负责的游戏网站：射命丸文的酉间坊纪行
虽然游戏为实验性作品还是不怎么成熟，但游戏本身还是非常好玩的东方 3D 剧情向同人游戏
游戏制作：鵺酱心酱制作委员会
正在逐步制作续作《寂静的故乡》重制版
希望大家能游玩他们的游戏和推荐，这是对同人创作的最好的帮助！

最后，祝大家⑨月⑨日琪露诺日快乐！
