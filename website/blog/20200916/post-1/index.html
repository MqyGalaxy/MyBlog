# 博客将取消 Gitee Pages 部署

为什么要取消？

因为最近将再一次改变网页格式，一个网站存在两个不同的域名对于超链接实在是太麻烦了
再加上本来就没钱买服务器等等各项需求品，动态的实现几乎很难
整理博客难上加难，况且本来该网站也没用多少交互功能，存在两个仓库也没有必要

原本 Gitee Pages 搭建的用途是解决大陆加载过慢 / 无法进入的问题，但是由于 Github 现在也可以访问，只要按照以下方式即可正常使用：

建议使用 Chrome 浏览器（最好不要使用 IE，谁用谁知道）
不要使用 QQ / 微信等软件自带浏览器进入，可能会导致加载过慢或者无法连接
在博客添加新域名后，可以忽视上一项
目前博客域名：https://mqygalaxy.github.io 将继续使用至新域名部署前
而：https://mqygalaxy.gitee.io 将在不久后停止使用

如若您在访问：gitee.io 域名时无法访问，请切换为：github.io

域名再次变动时将会发出公告
